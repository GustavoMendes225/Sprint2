Manual de Criatividade - Robo P2G3

1. Introdução

• Sobre o Robô Sumô P2G3

O Robô Sumô P2G3 é um projeto educacional envolvente, controlado por Arduino e equipado com sensores de fim de curso. Sua característica principal é uma arma para estourar balões, ativada pelos sensores. Além disso, possui LEDs personalizáveis, é alimentado por pilhas e controlado por um aplicativo Android via Bluetooth. Este robô proporciona experiências práticas e interativas, sendo uma ferramenta ideal para ensinar robótica e programação de forma criativa e estimulante.

• Objetivo do Manual

O objetivo deste manual é fornecer orientações detalhadas para montar, operar e explorar as capacidades criativas do Robô Sumô P2G3. Ele foi criado para facilitar o entendimento dos usuários sobre a estrutura do robô, suas funcionalidades e aplicações, estimulando a criatividade e promovendo uma experiência educativa envolvente. Além disso, o manual visa inspirar os usuários a explorar o mundo da robótica, encorajando a experimentação e inovação através do uso do P2G3 em diversas atividades educacionais e projetos criativos.

• Instruções Passo a Passo e Dicas de Montagem

Instruções Passo a Passo para Montagem do Robô Sumô P2G3:

Preparação dos Componentes:

Organize os componentes necessários: Arduino Mega, chaves de fim de curso, LEDs, fita de LEDs, chave liga-desliga, soquete de pilhas, motores de 5V, shield de motor, módulo Bluetooth, protoboard, transistores e resistores. Montagem da Estrutura:

Comece fixando os motores no chassi do robô. Conecte as rodas aos motores para permitir a movimentação. Posicione as chaves de fim de curso estrategicamente para interação com o ambiente. Fixe o Arduino Mega no chassi, garantindo fácil acesso às portas de conexão. Conexão dos Componentes Eletrônicos:

Utilize o shield de motor para conectar os motores ao Arduino, garantindo controle preciso dos movimentos. Conecte as chaves de fim de curso aos pinos digitais do Arduino para detecção de obstáculos. Conecte os LEDs e a fita de LEDs ao Arduino para iluminação personalizável. Configuração do Controle por Bluetooth:

Conecte o módulo Bluetooth ao Arduino e configure as conexões necessárias. Desenvolva um código para estabelecer a comunicação entre o robô e o aplicativo Android por meio do módulo Bluetooth. Alimentação e Liga-Desliga:

Conecte os motores e os componentes eletrônicos ao soquete de pilhas. Utilize a chave liga-desliga para controlar o fornecimento de energia ao robô. Testes e Calibração:

Verifique todas as conexões e faça testes para garantir que os motores, sensores e LEDs estejam funcionando corretamente. Calibre os sensores de fim de curso para assegurar detecção precisa de obstáculos. Desenvolvimento do Código:

Desenvolva o código Arduino para controlar os motores, sensores e a iluminação LED. Implemente lógica para o acionamento da arma de estourar balões com base nos dados dos sensores de fim de curso. Testes Finais:

Realize testes abrangentes para garantir que o robô esteja respondendo aos comandos do controle via Bluetooth de forma eficaz. Ajuste o código conforme necessário para otimizar o desempenho do robô. Personalização e Experimentação:

Experimente diferentes padrões de iluminação com os LEDs para criar efeitos visuais interessantes. Encoraje os usuários a experimentar e personalizar o código para desenvolver novos comportamentos e funcionalidades do robô.

3. Funcionalidades

• Sensores de Fim de Curso Eles servem para acionar a arma

• Controle da Arma de Estourar Balões O controle da arma é feito no codigo do arduino

• Iluminação com LEDs É possivel modificar o sistema de iluminação por meio do arduino, podendo ser personalizado da maneira que bem entender

4. Como Usar o Robô Sumô P2G3

• Conectar o Robô ao Aplicativo via Bluetooth Ative o Bluetooth no dispositivo Android. No aplicativo, selecione "Robô Sumô P2G3" para conectar automaticamente.

• Calibrar os Sensores de Fim de Curso No aplicativo, acesse a função de calibração. Siga as instruções para ajustar os sensores de fim de curso para detecção precisa de obstáculos.

• Acionar a Arma para Estourar Balões Utilize os controles no aplicativo para ativar a arma.

Os sensores de fim de curso acionam a arma quando detectam um obstáculo próximo.

• Personalizar a Iluminação com LEDs No aplicativo, escolha entre diferentes padrões de iluminação para os LEDs. Experimente e crie efeitos visuais personalizados para o robô.

6. Solução de Problemas Comuns

• O Robô Não Responde aos Comandos do Aplicativo

Verifique a conexão Bluetooth entre o dispositivo Android e o robô. Reative o Bluetooth se necessário. Certifique-se de que o robô está ligado e dentro do alcance Bluetooth. Reinicie o aplicativo, se preciso.

• A Arma não Estoura os Balões Adequadamente Calibre os sensores de fim de curso para uma detecção mais precisa de obstáculos. Confira a pressão do ar na arma. Ajuste conforme necessário para uma potência de estouro adequada.

• Problemas com a Iluminação LED Verifique as conexões dos LEDs. Certifique-se de que estão corretamente conectados aos pinos do Arduino.

Se a iluminação estiver inconsistente, verifique se há um problema elétrico ou se os LEDs estão danificados.

• Vida Útil da Pilha e Substituição Utilize pilhas de alta qualidade para uma vida útil prolongada.

Se o robô mostrar sinais de fraqueza, substitua imediatamente as pilhas. Pilhas recarregáveis podem ser uma opção para reduzir custos a longo prazo.
